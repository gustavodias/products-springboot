# 📑 Products - API RESTful

Esta API tem como finalidade o gerenciamento de produtos.
Por meio desta API o usuário poderar realizar CRUD basicos referente a produtos.

## ⚙️Tecnologias

- Java 17
- Spring Boot 3
- Spring Data JPA
- Spring Validation
- Spring Hateoas
- PostgresSQL
